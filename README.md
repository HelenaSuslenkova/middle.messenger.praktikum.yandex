### Как запустить:

- `yarn install` — установка пакетов,
- `npm run dev` - запуск в dev режиме
- `npm run build` - билд проекта
- `npm run start` - билд проекта и запуск nodejs сервера на порту 3000

См. [Package.json file](/package.json)
